# Changelog

Todos los cambios relevantes del proyecto se documentan en este archivo.

## [1.0.0] - 2026-01-07

### Added
- Integración completa con la Spotify Web API.
- Autenticación mediante OAuth 2.0.
- Endpoints REST para:
  - Información del usuario autenticado.
  - Artistas, álbumes y canciones.
- Documentación básica de la API.

### Changed
- Estructura final del proyecto orientada a una API REST.
- Mejora en la gestión de errores y respuestas HTTP.
